# Примеры relator

Сценарии показывают, как встроить библиотеку в **отчёты**, **тесты**, **покрытие**, **данные** и **агентные** пайплайны.

Рекомендуемый порядок изучения:

1. [00_showcase](00_showcase/) — **один файл-витрина**: все основные конструкции и пример готового `dist/showcase_report.md`.
2. [03_data_report](03_data_report/) — самый простой (только JSON + шаблон).
3. [01_tests](01_tests/) — отчёт по JSON результатов тестов.
4. [02_coverage](02_coverage/) — сводка по `coverage.json`.
5. [04_agents](04_agents/) — слоты под вывод агента / CI.

Статичные мини-примеры Markdown-вывода (таблица, список, цикл, слот) — в [../../docs/rendering-samples.md](../../docs/rendering-samples.md).

Запускайте `build_report.py` из каталога соответствующего примера (или укажите пути к шаблону и данным вручную). После генерации смотрите `dist/*.md`.
