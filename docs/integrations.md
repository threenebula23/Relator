# Интеграции

## Pydantic

Установка:

```bash
pip install "relator[pydantic]"
```

В контекст кладётся класс модели, инстанс или `TypeAdapter`. В шаблоне используйте `[[PYDANTIC.ИмяПеременной]]` и производные суффиксы `.TABLE`, `.JSON_SCHEMA`, `.EXAMPLE`.

Ошибка `TemplateError` с подсказкой установки появится, если пакет не установлен, а плейсхолдер встретился в шаблоне.

## SQLAlchemy: SQL и ORM

Установка:

```bash
pip install "relator[sqlalchemy]"
```

### `[[SQL.query]]`

- Если значение — `str`, оно оборачивается в fenced блок `sql`.
- Если значение — исполняемый объект SQLAlchemy, текст получается через `compile()` и выбранный диалект.

### `[[ORM.mytable.TABLE]]`

Подходит для документирования схемы БД: колонки, типы, nullable, PK, FK (первая FK в ячейке).

### `[[ORM.mytable.DDL]]`

Генерирует `CREATE TABLE` для диалекта `sql_dialect` конструктора `Template`.

## Безопасность

Плейсхолдеры **только отображают** SQL и метаданные в отчёте. Выполнение запросов к базе остаётся на стороне вашего приложения.
