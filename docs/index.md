# relator — документация

Библиотека **relator** (`reporting`) генерирует отчёты в **Markdown** и **HTML** из шаблонов с циклами, таблицами, списками, медиа, именованными слотами и опциональными интеграциями (Pydantic, SQLAlchemy).

## С чего начать

1. [Концепции](concepts.md) — короткий глоссарий и схема.
2. [Поток данных](data-flow.md) — как элементы шаблона связаны с `data`, слотами и JSON/CLI.
3. [Быстрый старт](getting-started.md) — установка и минимальный пример.
4. [Синтаксис шаблонов](template-syntax.md) и [справочник плейсхолдеров](placeholders.md) — у плейсхолдеров есть блоки **«Результат в отчёте»** с готовым Markdown/HTML.
5. [Примеры готового Markdown-вывода](rendering-samples.md) — таблицы, списки, циклы, слоты одним листом.
